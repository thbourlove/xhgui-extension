Xhgui-external

xhgui, xhprof, external, php

Download this project as a .zip file Download this project as a tar.gz file

What?

It's a easy external for xhgui.

How?

wget https://raw.github.com/thbourlove/xhgui-external/master/External.php

example

<?php
require "External.php";
$mongo = new MongoClient();
$xhprof = new \Thb\Xhgui\External($mongo->xhprof);
$xhprof->start();
print 1;
$xhprof->save();

With Composer

"require-dev": {
    "thbourlove/xhgui-external": "dev-master"
}

example

<?php
require "vendor/autoload.php";
$mongo = new MongoClient();
$xhprof = new \Thb\Xhgui\External($mongo->xhprof);
$xhprof->start();
print 1;
$xhprof->save();
